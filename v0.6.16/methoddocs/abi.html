class ethpm_types.abi.ABIType(*, name: str | None = None, type: str | ABIType, components: list['ABIType'] | None = None, internalType: str | None = None, **extra_data: Any)
property canonical_type: str

The low-level type recognized by the virtual machine. For example, a tuple is converted to comma-separated string of its components.

components: list['ABIType'] | None

A field of sub-types that makes up this type. Tuples and structs tend to have this field.

internal_type: str | None

Another name for the type. Sometimes, compilers are able to populate this field with the struct or enum name.

A dictionary of computed field names and their corresponding ComputedFieldInfo objects.

name: str | None

type: str | ABIType

The value-type, such as address or address[].
